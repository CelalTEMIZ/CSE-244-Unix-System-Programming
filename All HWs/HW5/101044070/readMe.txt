


--> Klasor icerisinde makefile bulunmaktadır. Klasor icerisine eristikten sonra make yapiniz.


--> Programin calistirilmasi icin de ./grephSh <aranacak string> <dosya> parametrelerini giriniz.


--> Odevde 
		   
		   - Semaphore,
		   - Fork,
		   - Thread,
		   - Ctrl + C signal handle,
		   - Timer,
		   - Message Queue,
		   - Shared Memory,						

		   	 mekanizmalari kullanilmistir.



--> MESSAGE QUEUE

--> Ödevde istendigi sekilde directory'ler arasinda message queue yapisi kullanilmistir.

    Directory sayilari message queue' ya yazilmaktadir ve toplam directory sayisi ekrana yazilirken message queue' dan alinmaktadir.

    Yazilan degerlerin dogrulugunun gorunumunu de comment olarak birakilmistir.




--> SHARED MEMORY 

--> Ödevde istendigi sekilde olusturdugum shared memory sayisi , directory sayisi kadardir. Sadece parametre olarak verilen directory buna dahil degildir.

	ProcessID, Dosya Adı,  aranan kelime ,  aranan kelimenin Hangi Satir ve Hangi Sutunda oldugu Shared Memory' e yazilmaktadir.

	Daha sonra log dosyasina bu veriler yazilirken, shared memory'den alinmaktadir.
